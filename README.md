# AI-Based District Health Infrastructure Stress Analysis

This project is an AI-based decision-support prototype that analyzes Indian government
healthcare and population datasets to identify districts under infrastructure stress.

## Key Features
- Population-to-PHC stress indicator
- Unsupervised learning (K-Means clustering)
- AI-based stress categorization
- Policy what-if simulation

## Tech Stack
Python, Pandas, Scikit-learn, Jupyter Notebook
